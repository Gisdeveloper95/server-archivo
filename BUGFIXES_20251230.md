### **2. Endpoint `folder_permissions` agregado**

**Ubicación:** `backend/files/views.py:1121-1144`

```python
@action(detail=False, methods=['get'], permission_classes=[IsAuthenticated])
def folder_permissions(self, request):
    """
    Obtener permisos detallados de un directorio
    """
    path = request.query_params.get('path', '')
    user = request.user

    try:
        # Obtener permisos detallados
        permissions = PermissionService.get_path_permissions_detail(user, path)

        # Agregar información adicional
        permissions['path'] = path
        permissions['user_role'] = user.role

        return Response(permissions)
    except Exception as e:
        return Response(
            {'error': f'Error obteniendo permisos: {str(e)}'},
            status=status.HTTP_500_INTERNAL_SERVER_ERROR
        )
```

**Función:**
- Retorna permisos detallados de un directorio
- Usa `PermissionService.get_path_permissions_detail()`
- Incluye: `can_read`, `can_write`, `can_delete`, `can_create_directories`, `read_only_mode`

**Respuesta:**
```json
{
  "can_read": true,
  "can_write": true,
  "can_delete": false,
  "can_create_directories": true,
  "read_only_mode": false,
  "path": "05_grup_trab/11_gest_info/2025",
  "user_role": "consultation_edit"
}
```

**Uso:**
```bash
GET /api/file-ops/folder_permissions?path=ruta/a/directorio
```

### **3. Endpoint `validate_name` agregado**

**Ubicación:** `backend/files/views.py:1146-1175`

```python
@action(detail=False, methods=['post'], permission_classes=[IsAuthenticated])
def validate_name(self, request):
    """
    Validar nombre de archivo/directorio antes de crear/renombrar
    """
    name = request.data.get('name')
    current_path = request.data.get('current_path', '')
    extension = request.data.get('extension', '')

    if not name:
        return Response(
            {'error': 'El nombre es requerido'},
            status=status.HTTP_400_BAD_REQUEST
        )

    user = request.user

    try:
        # Usar el servicio de validación
        validation = PathValidationService.validate_full_creation(current_path, name, user)

        return Response({
            'valid': validation['valid'],
            'errors': validation['errors'],
            'warnings': validation.get('warnings', []),
            'name': name
        })
    except Exception as e:
        return Response(
            {'error': f'Error en validación: {str(e)}'},
            status=status.HTTP_500_INTERNAL_SERVER_ERROR
        )
```

**Función:**
- Valida nombre ANTES de crear/renombrar
- Verifica:
  - Caracteres inválidos
  - Longitud máxima
  - Diccionario (palabras permitidas)
  - Extensión .gdb
  - Mayúsculas/minúsculas
  - Longitud total del path

**Request:**
```json
{
  "name": "andres_osorio_6",
  "current_path": "05_grup_trab/11_gest_info/2025/06_arch",
  "extension": ""
}
```

**Respuesta:**
```json
{
  "valid": true,
  "errors": [],
  "warnings": [],
  "name": "andres_osorio_6"
}
```

**Uso:**
```bash
POST /api/file-ops/validate-name
Content-Type: application/json

{
  "name": "nuevo_nombre",
  "current_path": "ruta/actual",
  "extension": ".txt"
}
```
