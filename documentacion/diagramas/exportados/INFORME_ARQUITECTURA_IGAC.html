1. Resumen Ejecutivo

1.1 Introduccion

El Sistema de Gestion de Archivos NetApp es una aplicacion web empresarial desarrollada para el Instituto Geografico Agustin Codazzi (IGAC), disenada para centralizar y optimizar la gestion de archivos almacenados en un servidor NAS NetApp. El sistema proporciona una interfaz moderna y segura para que los funcionarios de la entidad puedan navegar, buscar, subir, descargar y gestionar archivos de manera eficiente.

1.2 Objetivos del Sistema

  • Centralizacion: Unificar el acceso a archivos dispersos en el NAS NetApp
  • Control de Acceso: Implementar permisos granulares por usuario y ruta
  • Nomenclatura IGAC: Validar y sugerir nombres de archivos segun normas institucionales
  • Auditoria: Registrar todas las operaciones para trazabilidad
  • Colaboracion: Facilitar el intercambio de archivos mediante enlaces compartidos
  • Inteligencia Artificial: Asistir en el renombramiento de archivos con IA

1.3 Alcance

El sistema abarca las siguientes funcionalidades principales:

Modulo Funcionalidades

7.2 Instrucciones de Despliegue

Importante: Antes de desplegar, asegurese de tener acceso al NAS NetApp y de configurar las credenciales de montaje CIFS/SMB correctamente.
  1. Clonar el repositorio del proyecto
  2. Configurar archivo .env con credenciales
  3. Montar el share NAS en /mnt/repositorio
  4. Ejecutar docker-compose up -d
  5. Aplicar migraciones: docker-compose exec backend python manage.py migrate
  6. Crear superusuario: docker-compose exec backend python manage.py createsuperuser
  7. Acceder a https://gestionarchivo.igac.gov.co
